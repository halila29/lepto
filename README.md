direct formal requests to the Kelantan State Health Department, Kota Bharu,
Kelantan, Malaysia. Aggregated subdistrict-level summary data may be requested
from the corresponding author subject to approval from the relevant data custodians.

## Reproducibility Instructions

### Step 1: R Environment

R version 4.3.0 or later is required.

```r
install.packages(c(
  "tidyverse", "sf", "spdep", "gamlss", "gamlss.add", "gamlss.dist",
  "car", "ggplot2", "viridis", "RColorBrewer", "knitr", "kableExtra",
  "purrr", "moments", "stringr", "here", "patchwork", "writexl"
))
```

See `sessionInfo.txt` for the exact package versions used in the manuscript analysis.

### Step 2: Data Setup

Place the required data files before running any script:

```
data/Leptos.rds                        # individual leptospirosis records (restricted)
data/LAPORAN_BANJIR.rds                # flood event records (restricted)
data/mean_population_by_mukim.csv      # mean population by subdistrict
data/cases_by_mukim_with_climate.rds   # aggregated dataset for Scripts 02-04
shapefiles/kelantan.shp                # obtain from DOSM / OpenDOSM
shapefiles/kelantan.dbf
shapefiles/kelantan.shx
shapefiles/kelantan.prj
```

### Step 3: Run Scripts in Order

```r
source("R/01_data_preparation.R")        # ~5 min
source("R/02_descriptive_spatial.R")     # ~20 min  (999 permutations x 14 years)
source("R/03_spatial_gamlss.R")          # ~15 min
source("R/04_sensitivity_diagnostics.R") # ~10 min
```

All figures are saved to `outputs/figures/` and tables to `outputs/tables/`.

## Analysis Overview

### Script 01 — Data Preparation (`01_data_preparation.R`)

- Imports individual leptospirosis case records, flood reports, population, and shapefile
- Harmonizes subdistrict identifiers across all datasets (uppercase, trimmed)
- Transforms all spatial objects to UTM Zone 47N (EPSG:32647)
- Aggregates to subdistrict (mukim) level: total cases, demographic ratios,
  flood exposure proportions at 500 m, 1 km, and 2 km buffers
- Merges ERA5 temperature and mean annual population
- Computes annual incidence rate (per 100,000) over 13 years and 4 months
  (January 2011 - April 2024)
- **Output:** `data/cases_by_mukim_with_climate.rds`

### Script 02 — Descriptive Spatial Analysis (`02_descriptive_spatial.R`)

Reproduces **Table 1, Figure 4, Figure 5, Figure 6** from the manuscript.

- **Global Moran's I** for leptospirosis incidence and flood events per year
  (999-permutation Monte Carlo, one-sided, add-one p-value correction)
- **LISA (Local Indicators of Spatial Association):** High-High, Low-Low,
  High-Low, Low-High clusters at p <= 0.05; recurring hotspot classification
- **Getis-Ord Gi\*:** hotspot and coldspot maps at 95% and 99% confidence thresholds
- **Bivariate Moran's I:** spatial co-clustering of leptospirosis incidence
  and flood frequency, by year
- Uses a single stable queen contiguity weight matrix across all 14 years
  to ensure methodological consistency

### Script 03 — Spatial GAMLSS Modeling (`03_spatial_gamlss.R`)

Reproduces **Table 2, Table 3, Figure 7(a-c)** from the manuscript.

- Overdispersion diagnostic (variance-to-mean ratio)
- Stepwise covariate selection: `stepGAIC()` with BIC penalty (k = log n)
- VIF multicollinearity diagnostics (all VIF < 2.0)
- Eight candidate models compared: Poisson, NBI, ZINBI x non-spatial /
  spatial random effect (`random(mukim_id)`) / penalized B-spline (`pb()`)
- **Final model selected:** NBI + `random(mukim_id)` (Model 5)
  - Lowest RMSE (3.06), lowest MAE (2.20), pseudo-R2 = 0.999
  - Theoretically appropriate for overdispersed count data
- Subdistrict relative risk (RR) and exceedance probability P(RR > 1)
  computed from linear predictor and standard error
- Risk maps: raw RR, quintile classification (Q1-Q5), exceedance probability

### Script 04 — Sensitivity and Diagnostics (`04_sensitivity_diagnostics.R`)

Reproduces supplementary diagnostics in the manuscript.

- Residual diagnostics (histogram, Q-Q plot, GAMLSS worm plot) for both
  non-spatial and final spatial models
- Moran's I test on model residuals (pre- and post-spatial adjustment)
- Marginal effect plots for all four selected covariates
- Absolute prediction error maps (NBI spatial vs. Poisson spatial)
